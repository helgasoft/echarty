# Shiny: Create a proxy

Create a proxy for an existing chart in Shiny UI. It allows to add,
merge, delete elements to a chart without reloading it.

## Usage

``` r
ecs.proxy(id)
```

## Arguments

- id:

  Target chart id from the Shiny UI.

## Value

A proxy object to update the chart.
